[부트스트랩(BootStrap)]
- html, css, javascript를 이용해서 코드를 이미 만들어 놓은 라이브러리
- 사용자는 bootstrap 홈페이지에서 필요한 코드를 복사하여 사용
- 수정할 부분 : 링크 주소나, 텍스트 내용 등에 대한 수정만 하면 됨
- 단점 : 태그 내에 class 속성을 사용하기에,
개인이 생성한 class 속성의 이름과 중복되면 작동이 안되는 경우 있음 (class 이름 정의시 bootstrap 이름과 중복되지 않도록 유의)
: 코드가 좀 많음(부트스트랩 코드 수정 시 불편..)
